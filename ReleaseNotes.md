## What's New

### Features
- 3D fractal rendering with real-time ray marching (Mandelbulb, Menger Sponge, Julia 3D, Sierpinski, and more)
- Interactive camera controls: orbit, pan, zoom
- Customizable fractal parameters, lighting, and color settings
- Cross-platform: Windows, macOS, Linux, WebGPU (browser), Android

### Controls
- **Desktop**: Left-drag to orbit, right-drag to pan, scroll to zoom, `R` to reset, `Space` to toggle auto-rotate, `Esc` to toggle UI
- **Android**: Single-finger drag to orbit, two-finger pinch to zoom, two-finger drag to pan
- **Web**: Same as desktop (mouse controls)
